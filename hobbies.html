Youth Movement

I am a member (since 2017) and the secretary (since 2022) of a Youth Movement called La Jeunesse Oumontoise.

With the other members, we organize several events during the year to bring the small village of Mont-Sainte-Geneviève to life (it is near Lobbes so you for sure didn't know this village existed before you read its name here) and bring people together.

Development

I recently developed an application for my Youth Movement. We were organising a quiz of general culture and we needed a visual support to show the questions and the answers to the participants but we also needed a way to simplify the correction. So I ended up with a solution to do that by developing a complete web application that manages all the event easily.

For the technical side, the backend was developed using Node.js, for the frontend I used ReactJS and for the database, I used a MySQL database host locally on a Docker container, as the project will only be run locally.

If you want to see the code (which is certainly not perfect), you can check it here for the backend and here for the frontend.
